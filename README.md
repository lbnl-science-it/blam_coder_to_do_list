# To Do List Application
This repo is for the 2025 LBNL BLAM Coder Track, and contains the files for the project demonstrated in the Week 3 class. It includes the file `to_do_list_prd.md`, which is the Product Requirement Document for the example application. 

The two saved `json` files, 
* feature1_plan.json
* feature_2_plan.json

are saved `omni-engineer-lbl` chats, which you can load into your local instance if you would like to experiment with loading chats and using pre-existing context. You can also read them to see the prompt history. 

The slide deck for the example application is [here](https://docs.google.com/presentation/d/1Mdn-Qq-qyVCoazMVjoVa_7sjuyeDcatH12vWGTj77ZE/edit?usp=sharing) (Only accessible to LBNL users).
